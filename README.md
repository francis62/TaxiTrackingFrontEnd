## Proyecto

### ¿En qué consiste?

El proyecto consiste en el seguimiento de un taxi mediante coordenadas con el agregado de un chat utilizando websockets simulando que se esta hablando con el conductor

### ¿Que tecnologías se utilizan?

Para llevar a cabo el proyecto se utilizó Spring para el lado del Back End y React con Material para el lado del Front End

## ¿Como correrlo?

En el backend simplemente correr el archivo main de la aplicación llamado "TaxiTrackingApplication", se recomienda abrirlo con Intellij. 

Y del lado del Front End tenemos que correr los siguientes comandos en la consola:

```bash
$ npm install

$ npm run dev
```

Se recomienda abrir el proyecto Front End con Visual Studio Code.
